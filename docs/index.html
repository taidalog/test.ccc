ccc

ccc: customizable command countdown-timer.
コマンドを入力して使用する、カスタマイズ可能なカウントダウンタイマーです。プレゼンテーションなどに使えると思います。

使い方

  1. コマンドエリアにコマンドを入力する。
  2. Enterキーを押す。そうするとタイマーが起動する。Escapeキーで一時停止、Deleteキーでリセットができる。
詳細はREADMEを読んでください。

コマンド例

down 5:00
5分間カウントダウンします。
up 1:30
5分間カウントアップします。
down 5:00 up 1:30
5分間カウントダウンしたあと1分30秒間カウントアップします。
down 5:00 --color #222266 --background #aaccff
5分間カウントダウンします。文字色は濃い青、背景色は淡い青です。
down 5:00 --color #222266 --background #aaccff --message hello
5分間カウントダウンします。文字色は濃い青、背景色は淡い青で、タイマーの下には「hello」と表示します。
down 5:00 --pause up 2:00
5分間カウントダウンし、一時停止します。その後、ユーザがタイマーを開始したら2分間カウントアップします。
down 5:00 --alarm bell up 2:00 --alarm beep
5分間カウントダウンし、鈴の音を再生します。続けて2分間カウントアップし、ビープ音を再生します。

送信情報

  • 閲覧したページの URL
  • 閲覧したページのタイトル
  • 当サイトを閲覧した日時
  • 当サイトを閲覧した際の大まかな位置情報
  • 当サイトを閲覧した際の IP アドレス
  • 当サイトを閲覧した際のインターネット端末およびインターネットブラウザの種類

送信先

  • Google LLC
